# Nonparametric-Adaptive-EWMA-Scheme-via-Weighted-Cucconi-Statistic

This repository contains information and material for the Nonparametric Adaptive EWMA Scheme via Weighted Cucconi Statistic, a project whose purpose is to understand the prementioned statistical process control scheme and make it come to life with R. The paper was written by Wenjuan Lianga, Amitava Mukherjeec, Dongdong Xiangb, and Zhonghao Xu (see the link below).

# The paper can be found in: https://www.researchgate.net/publication/361137740_A_New_Nonparametric_Adaptive_EWMA_Procedures_for_Monitoring_Location_and_Scale_Shifts_via_Weighted_Cucconi_Statistic
